# Getting Google to Update Your Search Result

Your site is correctly configured with:
- **Favicon:** Company logo (chicken with chef hat) at multiple sizes (48px, 96px, 192px)
- **Meta description:** "Hot, Juicy, Made Fresh - Guelph's Favourite Fried Chicken"

Google can take **several days to a few weeks** to update search results. To speed this up:

## 1. Deploy Your Latest Changes
Make sure the updated files are live at https://www.zingerchicken.ca. Upload:
- `index.html` (and all other HTML files)
- `assets/favicon-48.png`
- `assets/favicon-96.png`
- `assets/favicon-192.png`
- `assets/favicon-company.png`

## 2. Request Re-Indexing in Google Search Console
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property (zingerchicken.ca) if not already added
3. Use **URL Inspection** → enter `https://www.zingerchicken.ca/`
4. Click **Request Indexing**

## 3. Clear Cache (Optional)
- Submit your sitemap: `https://www.zingerchicken.ca/sitemap.xml`
- Google will recrawl when it can (may take 1–2 weeks)

**Note:** Favicon updates often take longer than meta description updates. Be patient—Google caches these heavily.
